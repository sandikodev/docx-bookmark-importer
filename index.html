Features
Everything you need, nothing you don't.
📂

Drag & Drop

Drop any .docx file onto the popup or side panel — no file picker needed.

🧠

Smart Parsing

Extracts bookmarks from Word's internal hyperlink and heading structures automatically.

🚀

Direct Chrome Import

Imports bookmarks straight into Chrome via the native Bookmarks API — no copy-paste.

🗂️

Smart Mapping

Visual folder selector to choose exactly where bookmarks land in your Chrome tree.

🔍

Duplicate Detection

Automatically skips URLs already in your bookmarks so you never get duplicates.

💾

Download as HTML

Export as standard Netscape HTML for use in Firefox, Edge, or any browser.

📌

Side Panel

Pin it as a Chrome side panel for persistent access while you browse.

📜

History Log

Tracks recent import sessions with bookmark counts so you know what was imported.

🌐

i18n — EN & ID

Full English and Indonesian localization built-in.

🔒

100% Local

No server, no analytics, no data transmission. Everything runs in your browser.

Installation
Load the extension manually in Chrome (Manifest V3).
1

Download the latest release

Go to Releases and download docx-bookmark-importer-extension.zip.

2

Unzip the file

Extract the zip to a permanent folder on your computer (don't delete it after loading).

3

Open Chrome Extensions

Navigate to chrome://extensions/ and enable Developer mode (top-right toggle).

4

Load Unpacked

Click Load unpacked and select the extracted folder. The extension icon will appear in your toolbar.

5

Use it!

Click the extension icon, drag your .docx file, and hit Import to Chrome.

Contributing
Built with Svelte 5 + Vite. Contributions are welcome!
# 1. Clone the repo
git clone https://github.com/sandikodev/docx-bookmark-importer.git
cd docx-bookmark-importer

# 2. Install dependencies (requires pnpm)
pnpm install

# 3. Start dev server (for UI preview)
pnpm dev

# 4. Build the extension
pnpm build

# 5. Package as .zip for distribution
node package-extension.js
